Реализуйте алгоритм обхода игровой доски для получения информации о текущем состоянии игры.


Входные данные: переменная gameField - вложенный массив, хранит информацию о состоянии клеток поля:
‘x’ - в клетке поля стоит крестик;
‘o’ - в клетке поля стоит нолик;
null - пустая клетка.
 

Пример:
const gameField = [
[‘x’, ‘o’, null],
[‘x’, null, ‘o’],
[‘x’, ‘o’, ‘o’]
];

Как выглядит поле:

х	о	 
х	 	о
х	о	о
 
Выходные данные: сообщение о состоянии поля:

‘Крестики победили’ - на игровой доске есть 3 заполненные клетки с крестиками (по горизонтали или по вертикали или по диагонали)
‘Нолики победили’ - на игровой доске есть 3 заполненные клетки с ноликами (по горизонтали или по вертикали или по диагонали)
‘Ничья’ - на игровой доске нет 3 заполненных клеток с крестиками или ноликами (по горизонтали или по вертикали или по диагонали) или есть 3 клетки с крестиками и 3 клетки с ноликами
Пример:
‘Крестики победили’ - так как есть 3 заполненные клетки с крестиками по вертикали
